# gcode-3mf-plate-Combiner-and-multiplier
vibe coded combiner and multiplier for my personal use with .gcode.3mf project files that i regulary print on 3d printers that doesn't support Moonrakers Job queuing. You can upload whole project file, the tool will automatically gather every .gcode files that are available there and combine them into one, and if specified - also multiplies them.
